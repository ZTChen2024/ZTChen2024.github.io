逆波兰表达式

思路

  1. 遍历整个逆波兰式
  2. 遇到字符数字，使用Integer.valueOf()转换为数字压入栈
  3. 遇到+或*，弹出两个数x1和x2（不需要管先后顺序），执行x3 = x1 + X2 或(x1 * x2)，将x3压入栈
  4. 遇到-或/，弹出x1保存到tmp1,弹出x2保存到tmp2,执行（tmp3 = tmp2 - tmp1 ）或（tmp3 = tmp2 / tmp1）,将tmp3压入栈
  5. 遍历结束后，弹出最后一个栈元素即所求
