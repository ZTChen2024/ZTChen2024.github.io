数组

  1. 数组本身就是引用数据类型
  2. 创建数组对象会在内存中开辟一块连续的空间，占据空间的大小取决于数组的长度和数组中元素的个数
  3. 数组中的元素在内存中是一次紧密排列有序的
  4. 数字一旦初始化完成，其长度就是确定的，而数组的长度一旦确定就不能修改
  5. 可以直接通过下标索引的方式调用指定位置的元素，数组名就是首地址
  6. 数组的索引是从0开始

List

List集合实例化步骤

  1. 导入Java.util包

  2. 通过ArrayList类创建一个List对象

  3. 声明对象格式

    list<引用数据> list = new ArrayList<>();

常用方法

函数名 功能
add(Object element) 向列表尾部添加指定的元素
size() 返回列表中的元素个数
get(int index) 返回列表中指定位置的元素。index从0开始。
isEmpty() 判断列表是否包含元素
contains(Object 0) 包含指定的元素，则返回true
remove(int index) 移除列表中指定位置的元素，并返回被删除元素

基本特点

  1. List集合中元素有序并且可以重复
  2. 支持索引（索引从0开始）
  3. List容器中元素都对应一个整数型的序号记载其在容器中的位置
