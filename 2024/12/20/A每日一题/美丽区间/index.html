快慢指针

题目

自己思路

  1. 使用快慢指针，l标记区间左起始点，r标记区间右终点，sum标记当前区间总和，len表示最美区间长度（初始为n+1）
  2. 从r开始，向右滑动，一直扫描数组，当sum < 目标s时，一直实施sum += arr[r]
  3. 当sum>=s时，先更新len（取最小），开始遍历左起始点，向右滑动，直至sum<s
  4. 重复2和3，直至r = n 退出
  5. 如果len == n + 1说明不存在美丽区间，返回0，否则返回美丽区间的最短长度len
