Java基础类型与输入输出

发表于2024-12-08|更新于2024-12-08
|浏览量:

基础类型

类型 范围 默认值
int 范围是-2^31 到 2^31 - 1
long 范围是-2^63 到 2^63 - 1
float 单精度32位 0.0
double 单精度64位 0.0
boolean 2（true，false） false
char 16位Unicode字符

输入输出

  1. 导入java.util包
  2. 构造Scanner对象，参数为标准输入流System.in
  3. 使用next方法系列接受数据
nextInt(); 接受一个int整数
nextLong(); 接受long型整数
nextFloat(); 接受float型小数
nextDouble(); 接受double型小数
next(); 接受读取不带空格的字符串
nextLine(); 接受读取整段字符串
