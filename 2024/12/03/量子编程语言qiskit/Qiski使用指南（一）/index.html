一，环境配置

  1. 确定最新版本支持哪些 Python 版本https://wiki.python.org/moin/BeginnersGuide/Download)

    建议您使用Python 虚拟环境将 Qiskit 与其他应用程序分开。

    首先，创建一个仅安装 Python 的最小环境。

    接下来，激活您的新环境。

  2. 如果您的环境中尚未安装pip，请安装它pip list。Pip 是一个 Python 包管理器，可用于安装 Qiskit 和其他 Python 包。使用它来查看虚拟环境中的内容。在大多数 Python 环境中，pip 已安装。

  3. 安装 Qiskit SDK。如果您计划在量子硬件上运行作业，还请安装 Qiskit Runtime。

    如果您打算使用可视化功能或 Jupyter 笔记本，建议安装具有额外可视化支持的 Qiskit（'qiskit[visualization]'）。

  4. 如果您想使用刚刚安装的 Qiskit 包运行 Jupyter 笔记本，则需要在您的环境中安装 Jupyter。

    然后按如下方式打开你的笔记本：

保持最新版本

定期检查Qiskit 发行说明和Qiskit Runtime 发行说明以查看新版本。我们建议您经常更新qiskit和的要求qiskit-ibm-runtime，例如，将 中的版本更改requirements.txt为最新版本，然后运行pip install -r requirements.txt或适合您的依赖项管理工作流程的命令。

五，设置以使用 IBM Cloud

  1. 在使用 IBM Cloud 进行设置之前，请确保您在安装了 Qiskit SDK 和 Qiskit Runtime 的活动 Python 环境中工作。

  2. 如果您还没有 IBM Cloud 帐户，请从IBM Cloud 注册页面设置一个 IBM Cloud 帐户。

  3. 如果需要，请创建服务实例。打开IBM Cloud 实例页面。如果显示一个或多个实例，请继续执行下一步。否则，单击创建实例。创建实例时，您可以为其命名、标记、为其选择资源组以及选择性能策略。接下来，选中页面右下角的复选框以同意许可协议，然后单击创建。

    笔记

    如果您是需要为您的组织设置云端 Qiskit Runtime 的管理员，请参阅为组织规划 Qiskit Runtime。

  4. 查找您的访问凭证。

    1. 找到您的 API 密钥。从API 密钥页面，查看或创建您的 API 密钥，然后将其复制到安全位置，以便您可以用它进行身份验证。
    2. 找到您的云资源名称 (CRN)。打开实例页面并单击您的实例。在打开的页面中，单击图标以复制您的 CRN。将其保存在安全的位置，以便您可以将其用于身份验证。
